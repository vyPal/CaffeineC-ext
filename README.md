# CaffeineC

This extension adds support for the CaffeineC programming language in vscode.

## Features

- Basic syntax highlighting
- Live error checking

## Requirements

- Golang (tested with `1.21.3`)

## Known Issues

- Syntax highlighting is not complete
- Sometimes file turns white
